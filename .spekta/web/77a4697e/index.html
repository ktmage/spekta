spekta build
  • Web レンダラー
    • ページごとの HTML ファイルが生成される
    • index.html が最初のページにリダイレクトする
    • 404.html がルートにリダイレクトする
    • HTML にページタイトルが含まれる
    • HTML にサイドバーが含まれる
    • HTML にステップが自然言語で表示される
    • HTML にヘッダーのプロダクト名が含まれる
    • HTML にフッターが含まれる
    • HTML に see 参照がリンクとして含まれる
    • HTML に why 属性がコールアウトとして含まれる
  • Markdown レンダラー
    • ページごとの .md ファイルが生成される
    • Markdown にページタイトルが h1 として含まれる
    • Markdown にステップが番号付きリストとして含まれる
    • Markdown に see 参照がリンクとして含まれる
spekta doctor
  • Node.js のバージョンを表示する
  • プロジェクトディレクトリで実行した場合
    • .spekta.yml が見つかると表示する
    • spec ディレクトリが見つかると表示する
spekta watch
  • HTTP サーバーが起動してページを配信できる
  • spec ファイルを変更するとリビルドされる
  • SSE エンドポイントが利用可能である
