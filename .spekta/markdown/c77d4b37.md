# spekta watch

ファイル変更を監視し、自動リビルドとライブリロードを提供するコマンド。

## サーバーが起動している場合

### HTTP でページを配信できること

1. localhost にHTTPリクエストを送信する
2. ステータスコード 200 が返る

### SSE エンドポイントが利用可能であること

1. /__reload エンドポイントにリクエストを送信する
2. ステータスコード 200 が返る
3. Content-Type が text/event-stream である

## spec ファイルを変更した場合

### 自動でリビルドされること

1. テストファイルの内容を変更する
2. "Rebuild complete" の出力を待つ
3. 生成された HTML に変更内容が反映されている
