# spekta doctor

プロジェクトの環境と設定を診断するコマンド。

## Node.js のバージョンを表示すること

1. spekta doctor を実行する
2. 出力に Node.js バージョン情報が含まれる

## プロジェクトディレクトリで実行した場合

### .spekta.yml が見つかること

1. プロジェクトディレクトリで spekta doctor を実行する
2. 出力に .spekta.yml: found が含まれる

### spec ディレクトリが見つかること

1. プロジェクトディレクトリで spekta doctor を実行する
2. 出力に spec/features パスが含まれる
